# Portafolio Web - Lyn Studio

## Descripción

Este proyecto corresponde al desarrollo de un portafolio web personal para la marca **Lyn Studio**, creado como parte del parcial de Diseño Web.

El sitio presenta mi perfil profesional, servicios, portafolio y medios de contacto, integrando una identidad visual propia basada en colores rosados y tonos oscuros que reflejan creatividad, estilo y personalidad.

## Objetivo

Diseñar y desarrollar un portafolio web funcional que permita mostrar mis habilidades y proyectos en el área de Publicidad y Marketing Digital mediante una interfaz atractiva y organizada.

## Tecnologías Utilizadas

* HTML
* CSS
* JavaScript
* GitHub
* GitHub Pages

## Secciones del Sitio

### Inicio

Presentación visual de la marca Lyn Studio con acceso rápido al portafolio.

### Sobre mí

Información personal y profesional sobre Marilynn Díaz, estudiante de Publicidad y Marketing Digital.

### Servicios

* Creación de contenido creativo para redes sociales.
* Diseño de identidades visuales.
* Planeación de estrategias digitales.

### Portafolio

Galería de proyectos desarrollados en diseño gráfico y contenido visual.

### Contacto

Formulario de contacto y acceso a redes sociales.

## Características Implementadas

* Navegación intuitiva mediante menú principal.
* Identidad visual personalizada.
* Uso de imágenes y recursos gráficos propios.
* Organización del código en archivos HTML, CSS y JavaScript.
* Integración de enlaces a redes sociales.

## Componentes Diseñados

1. Barra de navegación personalizada.
2. Tarjetas de servicios.
3. Galería de proyectos.
4. Formulario de contacto.
5. Botones interactivos.
